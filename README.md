# wwwbk for Linux
Backup the /var/www folders (all JavaScript, TypeScript, and PHP code) into /var/www_backups folders...

### Download:
git clone https://github.com/Bob-586/wwwbk.git ~/wwwbk

## Goto folder of program :
$ cd ~/wwwbk
## Then Compile the program :
$ ./compile_me.sh

## To RUN :
$ wwwbk
